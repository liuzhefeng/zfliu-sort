##1.关于稳定性：
不稳定：快选堆希（快速排序、选择排序、堆排序、希尔排序）
稳    定：插冒归计基（简单插入排序、冒泡排序、归并排序、计数排序、基数排序）

##2.关于移动次数和关键字顺序无关的排序
顺口溜：一堆（堆排序）海龟（归并排序）选（选择排序）基（基数排序）友
